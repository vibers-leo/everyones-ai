# 🚀 모두의AI (Everyone's AI) 개발 계획서

## 1. 프로젝트 개요

- **목표:** 누구나 쉽고 즐겁게 AI를 활용해 게임을 만들 수 있도록 돕는 비영리 커뮤니티 플랫폼
- **핵심 가치:** Easy (쉬움), Fun (재미), Together (함께)
- **기술 스택:** Next.js (App Router), TypeScript, Firebase (Auth, Firestore, Storage)

---

## 2. 개발 로드맵 (Phase 별 진행)

### ✅ Phase 1: 기본 골격 및 퍼블리싱 (완료)

- [x] 프로젝트 초기 세팅 (Next.js, Globals CSS)
- [x] 디자인 시스템 구축 (Cute 3D Emoji Concept, Dark Mode)
- [x] 주요 페이지 퍼블리싱 (Home, About, Lab, Community, Login/Signup)
- [x] 반응형 Navbar 및 라우팅 구조

### 🚧 Phase 2: 사용자 인증 및 데이터베이스 연동 (우선 순위)

**목표:** 실제 회원가입/로그인 및 게시글 작성 기능 활성화

1. **Firebase 인증 (Auth)**
   - [ ] `.env` 환경변수 설정 가이드 및 적용
   - [ ] 회원가입/로그인/로그아웃 로직 검증
   - [ ] 소셜 로그인 (Google) 연동 확인
2. **커뮤니티 (Firestore)**
   - [ ] 게시글 쓰기: 실제 DB 저장 (제목, 내용, 카테고리, 작성자, 날짜)
   - [ ] 게시글 목록: DB에서 최신순 불러오기 및 페이지네이션/무한스크롤
   - [ ] 게시글 상세: DB 데이터 조회
   - [ ] 게시글 수정/삭제 기능 (본인 글만)

### 🔜 Phase 3: 인터랙션 및 마이페이지

**목표:** 사용자 간 소통 강화

1. **댓글 및 좋아요**
   - [ ] 게시글 하단 댓글 작성/삭제 기능
   - [ ] 게시글 '좋아요(❤️)' 카운팅 기능
2. **마이페이지 (프로필)**
   - [ ] 내 프로필 정보 수정 (닉네임, 한줄 소개)
   - [ ] 내가 쓴 글 / 좋아요 한 글 모아보기

### 🔜 Phase 4: 배움터 (Academy) & 쇼케이스

**목표:** 콘텐츠 소비 및 확산

1. **배움터 (Academy)**
   - [ ] 유튜브 강의 임베딩을 위한 비디오 카드 UI 구현
   - [ ] 난이도별(입문/중급) 강의 필터링
2. **쇼케이스 (Showcase)**
   - [ ] 사용자가 만든 게임(이미지/링크) 업로드 기능
   - [ ] 갤러리 형태의 그리드 뷰 구현

### 🔜 Phase 5: 관리자 및 배포

1. **관리자 기능 (Admin)**
   - [ ] 공지사항 작성 권한 관리 (운영자만)
   - [ ] 신고된 글 숨김 처리 등
2. **최적화 및 배포**
   - [ ] 메타 태그 (SEO) 설정 강화
   - [ ] Vercel 배포

---

## 3. 진행 방식

사용자님(USER)의 컨펌 후, **Phase 2부터 순차적으로** 코드를 구현합니다.
자리를 비우시는 동안 Phase 2(DB연동) -> Phase 3(댓글/마이페이지) -> Phase 4(쇼케이스)까지 쭉 개발을 진행하겠습니다.

**확인 후 '진행해' 라고 말씀해주시면 바로 시작합니다!**
